Haupteinstellüngen

Filter key

Gibt einen Schlüsselwort ein, über welches das Widget mit dem Filter ausgewertet wird. Allerdings für Filter Bar gibt es keinen Sinn einen FilterKey einzustellen, weil so kann es sich selbst ausfiltern.

Zeige in Views

Gerade für Bar-Widgets ist das relevant ein Bar-Widget in mehr als einem View zu zeigen. Hier kann man einstellen in welchen Views es noch angezeigt werden soll.
Es erscheint an der gleichen Stelle in anderem View.

Filter bar


Dieses Widget kann sehr schnell einen Set von Filterbuttons erstellen.
Mit dem Eigenschaft "FilterKey" jedes Widget kann zu einer Filtergruppe zugeordnet werden.
Filter Bar kann diese Gruppen auswählen. Falls keine Gruppe markiert ist, dann werden alle Gruppen angezeigt.
